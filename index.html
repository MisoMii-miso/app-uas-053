Tentang Website Ini

Mata kuliah ini membahas mengenai konsep dasar pengembangan aplikasi berbasis web. Fokus pembelajaran mencakup penyusunan struktur data menggunakan HTML, penghiasan layout arsitektur antarmuka dengan CSS, serta pemberian logika interaktif client-side menggunakan JavaScript.

Website ini mengusung tema "Katalog Buku". Aplikasi prototype ini dirancang untuk mendemonstrasikan integrasi layout dinamis, manipulasi data array lokal pada tabel transaksi, serta sistem keamanan validasi autentikasi dasar pada form login.
